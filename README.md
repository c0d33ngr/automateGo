## Install
Use [go mod](https://go.dev/ref/mod)  
```bash
sudo apt-get install xclip
git clone github.com/dreddsa5dies/automateGo
```

## Recording format: projects/07_01_regexpMail  
* 07 - chapter  
* 01 - project in the chapter  
* NAME - short title
